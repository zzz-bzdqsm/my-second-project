预约挂号平台系统（演示版）
后端：Spring Boot + MyBatis + MySQL + Redis + JWT；前端：静态页面（无 Node）。

注册 / 登录

登录成功后会把 token 存在 localStorage，并用于预约相关接口。

医生列表

接口：GET /api/doctors / GET /api/doctors/depts
ID 姓名 科室 职称 操作

排班与预约

- 排班查询：GET /api/schedules?doctorId=&date=YYYY-MM-DD（带 Redis 缓存）
- 创建预约：POST /api/appointments（需要 token，含防超卖原子扣减）
scheduleId timeSlot total available 操作

我的预约

接口：GET /api/appointments/my，取消：POST /api/appointments/{id}/cancel
appointmentId doctorId scheduleId status createdAt 操作
